# Professional Portfolio - Tron Sci-Fi Theme

Welcome to the professional portfolio of Varanasi Sai Srikara Sasank. This portfolio is designed with a premium, fully responsive, and highly interactive Tron Sci-Fi aesthetic.

## Features
- **Tron Sci-Fi Theme**: A deep space gray background with striking Electric Blue and Neon Orange accents.
- **Glassmorphism & Claymorphism**: Sleek frosted glass navigation panels and soft, elevated 3D cards.
- **Dynamic 3D Tilt Cards**: Skills and project cards realistically tilt and react to your mouse movements with full hardware acceleration.
- **Custom Hardware-Accelerated Cursor**: A glowing cyan dot and neon purple ring that smoothly follows the cursor and expands on interactive elements.
- **Intensity-Based Scroll Navigation**: Effortlessly switch between pages by firmly scrolling at the top or bottom boundaries of the site.
- **Interactive Contact Modal**: A beautiful glass modal featuring one-click copy-to-clipboard functionality for contact details.
- **Optimized Page Transitions**: Seamless, buttery-smooth page loading animations with GPU acceleration to prevent screen jitter.

## File Structure
- `index.html` - Homepage and Hero Section
- `about.html` - Personal Background and Information
- `skills.html` - Technical Expertise (Languages, AI/ML, Tools)
- `projects.html` - Featured Work and Applications
- `experience.html` - Professional Timeline
- `css/style.css` - Core design system, animations, and variables
- `js/script.js` - Interactive logic (Cursor, Tilt, Scroll Navigation, Modal, Transitions)
- `assets/` - Resumes and static assets

## Setup & Deployment
Simply open any of the HTML files in a modern web browser to view the portfolio. The project uses standard HTML, CSS, and Vanilla JavaScript, requiring no build tools or package managers.

To deploy, upload the entire folder to any static hosting provider like GitHub Pages, Vercel, or Netlify.
